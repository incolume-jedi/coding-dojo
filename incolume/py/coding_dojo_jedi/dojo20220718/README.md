# Coding Dojo

## Problema

**Pegar valor ASCII do caracter**

Escrever uma função get_char() que pega um numero e retorna o valor ASCII correspondente.

## Example

get_char(65)
return: 'A'

Para referência da tabela ASCII:  http://www.asciitable.com/
