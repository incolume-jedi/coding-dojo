## Problema

**Par ou Impar**

Crie uma função que receba um inteiro como argumento e retorne "Par" para números pares ou "Ímpar" para números ímpares.

## Exemplos

is_par(5) = 'Ímpar'
is_par(56) = 'Par'
is_par(567) = 'Ímpar'
is_par(568) = 'Par'
is_par(13) = 'Ímpar'
is_par(17) = 'Ímpar'
is_par(0) = 'Par'
