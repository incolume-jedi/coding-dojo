## Problema

**Remover exclamação**

Crie uma função para remover todo o sinal de exclamação presente no texto passado

## Exemplos

```
"Hello World!" -> "Hello World"
"Hello World!!!" -> "Hello World"
"Hi! Hello!" -> "Hi Hello"
"" -> ""
"Oh, no!!!" -> "Oh, no"
```
