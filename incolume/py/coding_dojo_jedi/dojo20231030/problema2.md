## Problema

**Inteiro Reverso**

Dado um inteiro x de 32-bit, retorne x com seus digitos invertidos. Se o reverso de x estiver fora do limite de 32 bits [-231, 231 - 1], então retorne 0.

Assuma que o ambiente permite armazenar inteiros 64-bit.

 ## Exemplos

Example 1:
Input: x = 123
Output: 321

Example 2:
Input: x = -123
Output: -321

Example 3:
Input: x = 120
Output: 21


Constraints:

-231 <= x <= 231 - 1
