## Problema

**Quais são os três menores números felizes consecutivos**

Um número é feliz se somando os quadrados de seus algarismos e iterando o processo seja possível chegar ao número 1.

feliz(7) == True

iteração|algarismos|soma
----|---|----
1|7^2 | 49
2|4^2 + 9^2|  97
3|9^2 + 7^2|130
4|1^2 + 3^2 + 0^2| 10
5|1^2 + 0^2| 1


feliz(19) == True

iteração|algarismos|soma
----|---|----
1|1^2 + 9^2 | 82
2|8^2 + 2^2|  68
3|6^2 + 8^2| 100
4|1^2 + 0^2 + 0^2| 1
