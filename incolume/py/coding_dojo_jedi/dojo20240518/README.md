## Problema

**Romeu e Julieta**

O clássico problema computacional, buzzfizz, em uma versão abrasileirada.

Onde a entrada são números inteiros, ou uma lista de inteiros, e para múltiplos:

- 3x retorna queijo;
- 5x retorna goiabada;
- 3x e 5x retorna "romeu e julieta"
