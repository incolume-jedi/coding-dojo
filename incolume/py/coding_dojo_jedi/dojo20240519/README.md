## Problema

**Convert Age to Days**

Crie uma função que receba uma idade em anos e retorne o valor em dias.

## Exemplos

```bash
calc_age(65) ➞ 23725
calc_age(0) ➞ 0
calc_age(20) ➞ 7300
```

    OBS:

    - Use 365 dias para o ano neste desafio;
    - Ignore o lapso temporal de dias entre o aniversário e data atual;
    - Os valores de entrada são restritos a inteiros positivos;
